* Our unique expense tracker app, NUSpend, is is tailored to students from National University of Singapore.
It helps users cultivate good financial management habits through tracking their expenses and setting periodical budgets.
With NUSpend, you can spend smart and save smart.
* This is a desktop application. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).
